# Pablo Castaneda Website

This is a professional website for Pablo Castaneda built with Next.js, React, and Tailwind CSS.

## Features

- Modern, responsive design
- Built with Next.js 16.0.0
- Tailwind CSS for styling
- TypeScript support
- Multiple sections (Hero, Features, Testimonials, FAQ, etc.)
- Professional components and UI elements

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Carlosrr0431/pablocastaneda.git
cd pablocastaneda
```

2. Install dependencies:
```bash
npm install --legacy-peer-deps
```
or
```bash
pnpm install
```

3. Run the development server:
```bash
npm run dev
```
or
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the website.

## Build for Production

```bash
npm run build
npm start
```

## Deployment

This project can be deployed on platforms like:
- Vercel (recommended)
- Netlify
